# 0.0.1 — 图片格式转换器 初版

## 范围

纯前端图片格式转换器 SPA。支持 PNG/JPEG/WebP/BMP 四种格式互转，全部在浏览器本地完成 Canvas API 处理。单文件和批量转换 + zip 打包下载。

## 来源

- owner_input: e000001
- requirement_defined: e000002
- owner_confirmation: e000003

## 需求摘要

1. 拖拽或选择图片文件上传
2. PNG/JPEG/WebP/BMP 互转
3. 原图与转换后预览对比，显示文件大小变化
4. 批量转换 + JSZip 打包下载
5. 全部浏览器本地完成，不上传服务器
6. 异常处理：非图片文件、损坏图片、同格式转换提示

## 标准交付基线

- **前端**：React 或 Vue SPA，Vite 构建，HTML5 Canvas API 格式转换，JSZip 打包，Tailwind CSS 或原生 CSS，响应式布局
- **后端**：无（纯前端静态部署）
- **数据库**：无
- **部署**：Nginx alias + try_files SPA 模式，`/projects/image-converter/`
- **测试**：Playwright 浏览器自动化测试
- **文档**：architecture.md、runbook.md、decisions/、iterations/0.0.1.md

## 业务模型

- ImageFile：用户上传的图片（PNG/JPEG/WebP/BMP）
- ConversionTask：单次转换（源文件→目标格式→Blob→大小对比）
- BatchJob：批量任务集合→zip 下载

## 产品流程

### 单文件转换
1. 拖拽/选择图片
2. 检测格式，展示原图预览
3. 选择目标格式
4. 浏览器 Canvas 转换
5. 左右对比预览 + 文件大小变化
6. 下载单张

### 批量转换
1. 拖拽/选择多个文件
2. 批量缩略图列表
3. 统一目标格式
4. 逐个转换
5. 批量结果列表（预览+大小变化）
6. 打包 zip 下载

### 异常路径
- 非图片→格式不支持提示
- 损坏图片→无法解析提示
- 源=目标→无需转换提示
- 超大图片→内存不足警告
